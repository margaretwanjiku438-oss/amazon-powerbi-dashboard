# 📊 Amazon Product Performance Dashboard — Power BI

## Project Overview
An interactive Power BI dashboard analyzing Amazon product data across 
pricing, discounts, ratings and category performance. Built to uncover 
actionable business insights for data-driven merchandising decisions.

## Tools Used
- Microsoft Power BI Desktop
- Microsoft Excel (data cleaning)
- DAX (Data Analysis Expressions)

## Dataset
- 1,465 rows of Amazon product data
- 1,351 unique products identified after deduplication
- Fields: Product Name, Category, Actual Price, Discounted Price, 
  Discount %, Rating, Rating Count

## Dashboard Features

| Visual | Insight |
|---|---|
| KPI Cards | Average price, discount %, rating & total products |
| Average Price by Category | Electronics leads in highest average price |
| Average Rating by Category | Office Products leads in customer satisfaction |
| Top 10 Highest Rated Products | Best performing products by customer rating |
| Discount vs Actual Price | Relationship between pricing and discount strategy |
| Category Slicer | Interactive filter across all visuals |

## Key Business Insights
- **Electronics** has the highest average actual price at KES 5.44K
- **Office Products** leads in average customer rating
- **Average discount** across all products is 48%
- **1,351 unique products** across 8 main categories
- USB cables and accessories dominate top rated products — 
  affordable practical items consistently outperform on ratings

## Skills Demonstrated
- Data importing and transformation in Power BI
- DAX measures and calculated fields
- Interactive dashboard design with slicers
- KPI card creation and formatting
- Scatter plot, bar chart and table visualizations
- Data cleaning and deduplication
- Business insight generation from raw data
